    • Data sience leverages knowledge and tools from multiple subdisciplines, which have for foundational pillars the following:

  • Mathematics: The basic building tools for the other disciplines, from linear algebra to analysis and combinatorics

  • Computer Science: Because of the large size of the datasets explored and the requirements for rapid processing, computer science is the main pillar of data science. Both applied (programming, databases etc...) and theoretical aspects (algorithm optimization, data structures...) are mobilized.

  • Statistics: Statistics provides the framework to analyze the data, from correct use of data cleaning and sampling methods, to characterization of the datasets (descriptive statistics), modelization (inferential statistics) and conclusions regarding the scope of the findings.

  • Machine learning: At the interface of computer science and statistics, machine learning provides algorithms that can "learn" from data to recognize patterns and build predictive models.

  • Data visualization: The various tools and techniques used to faithfully present the findings of an analysis to the 3D constrained human eyes.

    • As presented in more details here, a typical data science project will follow a 4 steps workflow, each of which involving specific tools and methods:

  • Data preparation: Acquiring, cleaning and storing the data. The main challenge is the heterogeneity in the nature and provenance of the data. Some useful tools: python, R, SQL, NoSQL (for big data the database ecosystem around Hadoop+HDFS or Spark)...

  • Data analysis: Descriptive analysis and modelization with machine learning algorithms or statistical models. Useful tools: scikit-learn, R, NLP libraries,graph libraries, weka, Mahout ...

  • Evaluation: Evaluation of the models. Requires understanding of the issues related to statistical inference, bias/variance, feature selection, algorithm optimization ... The Data Analysis and Evaluation steps are alternated until a satisfying solution is achieved

  • Communication: Dissemination and visualization of the information can take many forms, from reports, to online apps. Useful tools: javascript, D3.js, python, ,R, Tableau .../
